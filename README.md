# monkeypox 2025

Análise de dados do vírus monkeypox no Brasil, durante o período de 2022 a 2025.  
Dados disponibilizados pela OPAS (Organização Pan-America de Saúde) no site:

https://shiny.paho-phe.org/mpox/?origin=pb.
